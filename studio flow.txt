# 📖 Overview

StudioFlow Real Canvas Engine is a browser-based multimedia editing platform that renders all media directly onto an HTML5 Canvas.

Unlike a standard HTML video player, StudioFlow treats every imported media asset as an editable object on a timeline, enabling real-time composition, effects, positioning, resizing, and exporting.

No desktop installation is required.

Runs directly in modern web browsers.
